TDU MODDING TOOLS 1.4
---------------------

English Readers, please scroll down.

Cet outil permet:
+ l'échange des couleurs
+ la conversion des textures vers un format éditable (et vice versa)
+ l'édition de fichiers facilitée
+ et plus encore !

Nouveautés
    - General
        * parseur BNK corrigé ! (merci à Jeff). La plupart des BNK peuvent être ouverts.
        * correction de bugs divers
        * guide en ligne (en anglais) accessible depuis le menu principal ou via http://syfworld.free.fr/tdu/files/Guide-1.4/
        * corrections interface
    - Car Colors :
        * ajout de couleurs déchiffrées
    - MAP Tool :
        * fonction de détection des incohérences MAP/fichiers
        * correction d'une entrée incohérente
        * correction de toutes les entrées incohérentes
    - File Browser :
        * drag&drop de BNK dans le BNK Manager
        * gestion automatique de l'édition
        * backup/restore opérationnels
        * remplacement de fichiers en tronquant/remplissant si nécessaire
    - 2DB vers DDS :
        * extraction puis conversion multiple depuis un BNK

Note au sujet des couleurs : la manipulation des couleurs est tout à fait opérationnelle.
Seule restriction : toutes les couleurs ne sont pas affichées en clair (N/A).


N'hésitez pas à me parler de tout problème, bug, etc.... via MP sur le forum.
Merci.

This tool provides:
+ color exchange
+ texture converting to editable file types (and vice versa)
+ easier file editing
+ and more !

Changes
    - Main
        * BNK parser fixed ! (thx to Jeff). Most BNK can be opened.
        * some misc. bugs fixed
        * online guide available by the menu or via http://syfworld.free.fr/tdu/files/Guide-1.4/
        * gui fixed
    - Car Colors :
        * new colors and cars added
    - MAP Tool :
        * detects unlinked MAP/files sizes 
        * incorrect entry fix
        * all incorrect entries fix
    - File Browser :
        * BNK file drag&drop in BNK Manager
        * automatic file editing
        * backup/restore operational
        * packed files replacement by truncating/filling when necessary (Jeff's Method)
    - 2DB to DDS :
        * extraction then multiple conversion from a BNK file

About colors : car colors editor is fully operational.
Only restriction for this point : some of the colors and car models are not clearly displayed (N/A).
