TDU MODDING TOOLS 1.3
---------------------

English Readers, please scroll down

Cet outil permet:
+ l'échange des couleurs
+ la conversion des textures vers un format éditable (et vice versa)
+ et plus encore !

Nouveautés
	- recherche dans une liste : recherche globale possible
	- ajout de couleurs déchiffrées
	- DDS vers 2DB et 2DB vers DDS :
		* sélection du fichier source par glisser/déplacer dans la case correspondante
		* le nom du fichier cible est renseigné automatiquement après sélection du fichier source.
		* effacement des champs après conversion réussie
	- BNK Tool : harmonisation de l'interface avec les autres modules
	- File Browser
		* Possibilité d'ouvrir les MAP / BNK / 2DB (éditeur DDS requis) / TDUCarColors.*
		* Fonction de backup des BNK
		* Extraction de fichiers contenus dans le BNK (qques bugs à corriger sur certains)
		* Un seul browser à la fois
		* mémorisation du dernier emplacement d'extraction
	- Paramètres : emplacement de TDU


Note au sujet des couleurs : la manipulation des couleurs est tout à fait opérationnelle.
Seule restriction : toutes les couleurs ne sont pas affichées en clair (N/A). Mais de nouvelles couleurs apparaîtront à la prochaine version.


N'hésitez pas à me parler de tout problème, bug, etc.... via MP sur le forum.
Merci.

ENGLISH

This tool provides:
+ color exchange
+ texture converting to editable file types (and vice versa)
+ and more !

Changes
	- searching in a list : global search available
	- more colors have appeared in Colors Editor !
	- DDS > 2DB and 2DB < DDS :
		* source file can be drag&dropped into the corresponding field
		* target file name is automatically set after the source is selected
		* fields are cleant after a succesful operation
	- BNK Tool : graphical interface made similar to other modules
	- File Browser
		* Able to open files : .MAP / .BNK / .2DB (DDS editor required) / TDUCarColors.*
		* BNK Backup function
		* Extraction of packed files in BNKs (some bugs remain though)
		* last extract location is kept into memory, as long as browser window remains opened
	- Parameters : TDU location setting


About colors : car colors editor is fully operational.
Only restriction for this point : some of the colors and car models are not clearly displayed (N/A). But new colors will be avaialble in next version.
