TDU MODDING TOOLS 1.9
---------------------

English Readers, please scroll down.

Nouveautés
    - Général :
        * Correction de bugs divers
        * Quelques modification visuelles mineures
        * Les fichiers temporaires sont maintenant nettoyés uniquement à la fermeture
        * Le mode d'affichage basique/avancé du panneau est automatiquement sauvegardé
    - Lanceur :
		* Toutes les fonctionnalités sont réunies en un seul bouton
    - File Browser :
		* La visibilité du BNK Manager et de la liste des tâches est automatiquement sauvegardée 
		* Affichage des propriétés de fichiers (clic droit > Properties...)
		* Bnk Manager: Ajout d'un bouton 'Load...' pour charger directement un fichier BNK
	- Camera&IK Fix (Nouveau !) :
		* Permet d'utiliser les caméras d'un autre véhicule pour conserver les positions préconisées par le moddeur
		* Permet également d'utiliser les infos de localisation (IK) d'un autre véhicule (mains du conducteur, ceintures de sécurité, etc)
    - Names Editor :
		* Inclut la dernière liste de noms (r18-3)
    - Settings :
		* Fonction de réparation de la base de registres (emplacement TDU, version du jeu)
		* Fonction de suppression du fichier radial.cdb
	- Patch Editor :
		* Proposition de valeurs pour entrer certains paramètres
		* Gestion des patchs installeurs/désinstalleurs
		* Nouvelles instructions : changeCameraSet / changeIKSet
	- ModAndPlay! :
		* Gestion des patchs installeurs/désinstalleurs.

N'hésitez pas à me parler de tout problème, bug, etc.... sur http://forum.tdu-central.com.
Merci.


Djey.

ENGLISH
-------

Changes
    - Main:
        * Many bug fixes
        * A few minor visual enhancements
        * Temporary files automatically cleaned only when closing
        * Basic/Advanced display mode in main panel is saved into settings 
    - Launcher:
		* All features now gathered in one single button
    - File Browser:
		* BNK Manager and Edit Tasks visibilities are saved into settings
		* File properties can be displayed (right-click, Properties)
		* Bnk Manager: new 'Load...' button to directly open a BNK file
	- Camera&IK Fix (new !):
		* Allows to use cameras from another vehicle to maintain modder's recommended positions
		* Also permits to use placing information (IK) from another vehicle (driver's hands, seatbelts, etc.)
    - Names Editor:
		* Includes latest name list (r18-3)
    - Settings:
		* Windows registry fix feature (TDU's main folder, game version)
		* Ability to delete radial.cdb file, on demand
	- Patch Editor:
		* When setting some parameters, known values are proposed
		* Install/uninstall patches support
		* New instructions: changeCameraSet / changeIKSet
	- ModAndPlay!:
		* Install/uninstall patches support.
		
As always, feel free to contact me if you encounter problems, bugs, ... @ http://forum.tdu-central.com.
Thanks.


Djey.
