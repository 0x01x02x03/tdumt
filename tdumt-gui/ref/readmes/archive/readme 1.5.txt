Changes
    - Main
        * Many bug fixes
        * BNK Tool disabled, please use File Browser now
        * New module : Database Editor (view mode + edit mode : File Browser > Edit your Database files)
        * Settings :
            + TDUMT can start with the module of your choice, at the same time
            + Database default language for DB Editor
        * Some annoying messages deleted
    - MAP Tool :
        * GUI was cleaned
        * Performance improvements
        * New files supported
    - File Browser :
        * Advanced replacing finally supported (files of different sizes)
        * File replacement by drag&drop in BNK Manager
        * BNK opening by drag&drop in BNK Manager
        * New keyboard shortcuts
        * New "MAP Details" modules with checking and fixing of file sizes (select a file then hit CTRL+M to activate)
    - DDS to 2DB :
        * GUI has evolved as a new convert method is going to appear soon
