TDU MODDING TOOLS 1.8
---------------------

English Readers, please scroll down.

Nouveautés
    - Général :
        * Correction de bugs divers
        * Ce logiciel nécessite d'être administrateur du PC
        * Accès au forum TDU Central depuis le menu d'aide
        * 'A propos de' : image perso et version de TDU affichées
        * Activation possible du mode debug, fournissant des informations dans le fichier "ModdingToolsDebug.log"
    - Launcher:
		* Support des commandes personnalisées avant / juste après lancement de TDU
    - Names Editor :
		* 2e partie, support des marques
		* Inclut la dernière mise à jour de la table de noms
    - File Browser :
		* Quelques modifications visuelles
		* Edition de fichiers :
			> les fichiers modifiés apparaissent en bleu dans la liste de tâches
			> double clic sur une tâche ouvre à nouveau le fichier en cours d'édition
    - Patch Editor :
		* Seulement accessible en vue avancée
		* Nouvelle instruction : changeRimName, permettant de changer le nom des jantes
		* Nouvelle instruction : changeBrandName, permettant de changer le nom d'une marque de vehicule ou vêtements
		* La console est mise à jour de façon instantanée
	- Database Editor :
		* Quelques modifications visuelles
		* Les données par topic peuvent être directement modifiées à présent (expérimental)

N'hésitez pas à me parler de tout problème, bug, etc.... sur http://forum.tdu-central.com.
Merci.


Djey.

ENGLISH

Changes
    - Main:
        * Many bug fixes
        * This software requires admin privileges to work
        * Access to TDU Central forum by Help menu
        * "About" dialog box: Custom picture :) and TDU version now displayed
		* Debug mode can be enabled, providing information in "ModdingToolsDebug.log" file        
    - Launcher:
		* Custom command support before / just after TDU launch
    - Names Editor:
		* 2nd part, brand name support
		* Includes latest update of Name reference table
    - File Browser:
		* A few visual changes
		* File editing:
			> modified files appear in blue in task list
			> double-clicking on a task opens edited file again
    - Patch Editor:
		* Only available in advanced view
		* New instruction: changeRimName, allowing to update rims database
		* New instruction: changeBrandName, allowing to update vehicle and wear brand names in database
		* Console view is automatically updated
		* A few messages fixed (sorry for bad english) 
	- Database Editor:
		* A few visual changes
		* Per-topic data can be directly modified now (experimental)
		
As always, feel free to contact me if you encounter problems, bugs, ... @ http://forum.tdu-central.com.
Thanks.


Djey.
