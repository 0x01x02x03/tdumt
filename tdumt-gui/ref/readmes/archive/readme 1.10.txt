TDU MODDING TOOLS 1.10
----------------------

English Readers, please scroll down.

Nouveautés
    - Général :
        * 100% compatible avec le patch communautaire 1.67a !
        * Correction de bugs divers
        * Fenêtre principale réorganisée
        * Les fichiers temporaires sont maintenant nettoyés uniquement à la fermeture
        * Le mode d'affichage basique/avancé du panneau est automatiquement sauvegardé
        * La gestion des BNK est en cours de refonte > Amélioration de fiabilité
        * Correction de bugs liés aux mipmaps dans la conversion 2DB<>DDS.
    - Vehicle Manager :
		* Nouvel outil permettant d'installer/gérer les véhicules de TDU et certaines caractéristiques
			¤ Installation de mods par type sur le véhicule de son choix (3D, jantes, compteurs, sons)
			¤ Modification de la marque et du modèle de véhicule
			¤ Modification des points de vente et V-RENT pour un véhicule
			¤ Modification du magasin de tuning pour un véhicule
			¤ Modification de la transmission (traction, propulsion, 4 roues).
	- File Browser :
		* Nouveau réglage (setting) pour définir l'emplacement initial des fichiers de remplacement (dossier courant ou personnalisé).
    - Names Editor :
		* Inclut la dernière liste de noms (r19-4).
	- Camera & IK Wizard :
		* Message d'erreur plus clair lors d'une erreur de chargement de la base de données
		* Inclut la dernière liste de slots (r19-1).
	- Patching :
		* Nouvelles variables : pointant vers les emplacements suivants
			+ Bnk\Vehicules, Rim
			+ Sound\Vehicules
			+ Frontend\HiRes\Gauges
			+ Frontend\LowRes\Gauges
			+ Frontend\AllRes
		* Une barre de progression apparaît pendant l'exécution du patch..

Crédits :

	- Tests: big-t991 / 2CVSUPERGT / Djey / Reilsss / Thunderhawk / Tool831.

Changes
    - Main:
        * Fully compatible with community patch 1.67a !
        * Misc. bug fixes
        * Re-arranged main window
        * Temporary  files are now cleaned when exiting only
        * Basic/advanced display mode of main panel is automatically saved
        * BNK processing is currently being rewritten > reliability enhancements
        * Mipmap bugs fixed in 2DB<>DDS conversion.
    - Vehicle Manager:
		* New tool allowing installing and managing TDU vehicles and a few characteristics
			¤ Mod installing according to part kind over vehicle of your choice (3D, rims, huds, sounds)
			¤ Modifying brand, model and vehicle version (aims at replacing Names Editor)
			¤ Setting selling and V-Rent spots
			¤ Setting tuning dealer
			¤ Modifying wheel drive (FWD, RWD, AWD).
	- File Browser:
		* New setting to define initial location of replacing files (current folder or custom one).
    - Names Editor:
		* Includes latest name list (r19-4). There won't be any more update as Names Editor will disappear in next version.
	- Camera & IK Wizard:
		* Clearer error message when error occurs while database loading
		* Includes latest slot list (r19-1)
	- Patching:
		* New variables: covering following locations
			+ Bnk\Vehicules, Rim
			+ Sound\Vehicules
			+ Frontend\HiRes\Gauges
			+ Frontend\LowRes\Gauges
			+ Frontend\AllRes
		* A progress bar now appears while patch is running.
