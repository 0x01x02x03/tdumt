Nouveautés
    - Général :
        * Correction de bugs divers
        * Cette version n'integre plus le BNK Editor de Jeff Harris
        * Remplacement : la modification de taille est complètement opérationnelle. A utiliser donc.
        * Le dossier d'installation de TDU est déterminé automatiquement
        * Nouvel assistant : Patch Editor
        * Nouvel assistant : Names Editor. 1e partie, support des noms de véhicules.
        * Aide en ligne : nouveau style. 
    - File Browser :
		* Suppression des Map Details. Passez au magic Map !
		* La suppression de fichiers passe maintenant par la corbeille
    - DDS vers 2DB :
		* Interface remaniée : bouton GO mis en évidence
        * (experimental) permet de générer un 2DB sans nécessité de l'original
    - 2DB vers DDS :
		* Interface remaniée : bouton GO mis en évidence
		* Mode multiple : le dossier des 2DB est affiché sous forme de lien cliquable.

Changes
    - Main:
        * Many bug fixes
        * This release does not integrate Jeff Harris' BNK Editor anymore
        * Replacing: 'size modifying' is now fully supported. So please use it.
        * TDU's install folder is now automatically detected
        * New wizard: Patch Editor
        * New wizard: Names Editor. 1st part, vehicle name support.
        * Online help : new style. 
    - File Browser:
        * File Details removed. Use magic Map instead!
        * Deleted files are now moved to the recycle bin
    - DDS to 2DB:
        * Modified interface: GO button more visible
        * (beta) ability to convert without original 2DB file 
    - 2DB to DDS:
		* Modified interface: GO button more visible
		* "multiple conversion" mode : folder where 2DB are stored is now displayed as clickable link.

As always, feel free to contact me if you encounter problems, bugs, ... by sending a private message in the forums.
