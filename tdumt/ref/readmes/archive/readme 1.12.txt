		* Mini-moteur de recherche de fichiers empaquetés (vue à plat seulement).

ENGLISH
-------

Changes
    - Main:
        * Misc bugs fixed
        * 'Car Colors' tool removed. Please use Vehicle Manager>Colors instead
        * New BNK management.
    - File Browser:
		* Minor visual enhancements
		* Old replacing method (maintainSize) removed
		* Blue indicator (modified files) removed
		* Packed file renaming (except extension)
		* Replacing and renaming in one operation
		* Bnk Manager button labels changed
		* 2 views for BNK contents (flat / hierarchical)
		* Tiny search engine for packed files (flat view only).		        
	- Patching :
		* 4 new instructions:
			+ bnkRemap/bnkUnmap
			+ installPackedFile/uninstallPackedFile
		* New variable: bnkLevelHawaiPath.
	- Vehicle Manager :
		* Physics>Gearbox: added control on gear ratios to ensure required values have been entered
		* Dealers>Vehicle List: ability to move a vehicle up/down over all slots
		* Physics now editable: Transmission Primacy-gearbox inertia-engine inertia.

Credits:
	- Design and programming: 2CVSUPERGT / Djey
	- Testing: 2CVSUPERGT / philippeano / Reilsss... 


Thanks to people making all those things real.


Djey.
