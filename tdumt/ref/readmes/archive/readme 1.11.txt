TDU MODDING TOOLS 1.11
----------------------

English Readers, please scroll down.

Nouveautés
    - Général :
        * 100% compatible avec le patch communautaire 1.67 !
        * Correction de bugs divers
        * Les outils avancés n'apparaissent plus en mode basique
        * Outil 'Custom Names' retiré. Utilisez le Vehicle Manager>Datasheet à la place
        * L'outil 'Cameras & IK Fix' est maintenant disponible en mode avancé uniquement
        * Tous les jeux de cameras et IKs de TDU sont disponibles (attention, certains peuvent ne pas fonctionner).
    - DB Resources :
		* Correction de bug sur l'affichage de certains identifiants de ressources
		* Chargement de fichier re-codé et optimisé.
    - Vehicle Manager :
		* Correction de bug sur l'installation de jantes AR
		* Mercedes SLK AMG 55 Performance Package maintenant modifiable
		* Seuls les véhicules conduisibles peuvent être édités
		* Possibilité de visualiser les véhicules par leur nom moddé
		* Modification du type de boite de vitesse (séquentielle, levier, semi-auto, auto)
		* Modification du nombre de vitesses et des coefs de transmission
		* Modification de l'emplacement moteur (arrière, central, avant)
		* Modification du régime moteur maxi
		* Désactivation/réactivation de l'effet sonore turbo
		* Modification de la hauteur de caisse AV/AR
		* modification de la hauteur de suspensions AV/AR
		* Intégre un module pour Caméra+IK Swap.
	- Patching :
		* Nouvelle option pour l'éditeur de patchs: nettoyage automatique du rapport
		* Nouvelles instructions :
			+ installFile
			+ uninstallFile
			+ checkPatch
			+ updateDatabase
			+ updateResource
		* Les instructions inutilisées ont été supprimées
		* Nouvelle commande pour dupliquer une instruction
		* Possibilité de créer un patch de désinstallation depuis un patch d'install
		* Deux déploiements: installeur et désinstalleur
		* Mode d'emploi détaillé en cours de réalisation par 2CVSUPERGT, avec accès via la barre de menus.

Crédits :
	- Conception et programmation: 2CVSUPERGT / Djey
	- Tests: 2CVSUPERGT / Djey / philippeapo / TDU-Piet / Reilsss. 


Merci à tous.


Djey.

ENGLISH
-------

Changes
    - Main:
        * Fully compatible wih community patch 1.67!
        * Misc bugs fixed
        * Advanced tools do not appear in basic mode anymore
        * 'Custom Names' tool removed. Please use Manager>Datasheet instead
        * 'Cameras & IK Fix' tool only available in advanced mode
        * All TDU cameras and IK sets are available for swapping (warning though, some may not work, please report).
    - DB Resources :
		* Fixed bug about displaying resource identifiers
		* File loading re-coded and optimized.
    - Vehicle Manager:
		* Fixed bug about rear rims installing
		* Mercedes SLK AMG 55 Performance Package now modifiable
		* Only driveable vehicles cen be modified
		* Ability to display vehicles by modded name (changed brand/model/version)
		* Gearbox type changing (sequential, mechanical, semi-automatical, automatical)
		* Gears count and ratios editing. *Warning, changes car behaviour*
		* Engine location changing (rear, center, front). *Effect on sound source only*
		* Redline engine changing (RPM). *Does not enhance performance*
		* Turbo enabling/disabling. *Effect on turbo sound FX only*
		* Front/rear ride height changing. *Warning, changes car behaviour*
		* Suspension height changing. *Warning, changes car behaviour*
		* Allows to swap Camera and IK.
	- Patching :
		* New Patch Editor setting: automatic report cleaning
		* New instructions:
			+ installFile
			+ uninstallFile
			+ checkPatch
			+ updateDatabase
			+ updateResource
		* Unused instruction have been removed
		* New command to duplicate selected instruction
		* Ability to create an uninstall patch from a finalized install patch
		* 2 deployments ways: installer/uninstaller
		* Detailed guide in progress by 2CVSUPERGT, which will be accessible via Patch Editor menubar.

Credits:
